# 项目分类总览

这份文档把当前仓库按用途分成几类，方便快速定位源码、数据、实验结果和归档内容。

## 1. 源码与执行脚本

这一类是项目的“可运行核心”。

- `scripts/train/`：训练入口
- `scripts/eval/`：评估与可视化
- `scripts/infer/`：推理脚本
- `scripts/data_prep/`：数据划分与预处理
- `scripts/nas/`：NAS 搜索、重训练、结果分析
- `scripts/benchmark/`：基准测试与模型对比
- `scripts/visualization/`：Grad-CAM 等可视化

## 2. 配置文件

这一类负责环境和运行参数。

- `configs/ase-jscc-gpu.yml`
- `configs/environment.yml`

## 3. 数据集

这一类是数据本体和数据划分结果。

- `data/AID/`
- `data/AID-train/`
- `data/AID-valid/`
- `data/AID-test/`
- `data/SoyaHealthVision/`
- `data/UCMerced_LandUse/`
- `data/UCMerced_LandUse_old（复件）/`
- `data/Rice crop diseases drone captured dataset/`

## 4. 文档与说明

这一类是项目说明、方法说明和图示。

- `docs/项目说明书.md`
- `docs/项目结构说明书.md`
- `docs/原始模型训练指南.md`
- `docs/模型基准测试指南.md`
- `docs/NAS代码快速开始.md`

- `docs/信道感知多目标NAS方案.md`
- `docs/dataset.txt`
- `docs/figures/`

## 5. 训练与实验结果

这一类是模型训练输出、搜索输出、日志和可视化结果。

### 5.1 权重文件

- `results/checkpoints/`
- 典型文件：
  - `results/checkpoints/best_classifier_attention_auto_AID_Combined_channel_ResNet18_150epoch_0.8.pth`
  - `results/checkpoints/classifier_attention_auto_UCMerced_LandUse_Combined_channel_ResNet18_150epoch_0.8.pth`

### 5.2 评估图与日志

- `results/logs/`
- 典型文件：
  - `results/logs/SoyaHealthVision_ResNet18/confusion_stage3_full_Combined_channel.png`
  - `results/logs/SoyaHealthVision_ResNet18/tsne_stage3_full_Combined_channel.png`

### 5.3 训练过程记录

- `results/runs/`
- 适合放：
  - TensorBoard 事件
  - 训练曲线
  - NAS 搜索过程
  - 重训练过程
  - 模型对比结果

### 5.4 当前已有的结果分组

- `results/runs/original_train/`：原始训练结果
- `results/runs/nas_search/`：NAS 搜索结果
- `results/runs/nas_retrain/`：NAS 重训练结果
- `results/runs/model_benchmark/`：模型基准测试结果
- `results/runs/nas_search_archive/`：归档的 NAS 搜索结果
- `results/runs/nas_search_formal/`：正式 NAS 搜索结果
- `results/checkpoints/`：通用最终权重
- `results/logs/`：图像化评估结果

### 5.5 额外实验目录

这些按数据集和实验条件命名的结果目录已经统一移动到 `results/archive/`：

- `results/archive/AID_150_combine_0.8/`
- `results/archive/Soya_1_combine_0.8/`
- `results/archive/Soya_10_combine_0.8/`
- `results/archive/Soya_120_combine_0.8/`
- `results/archive/UCMerced_LandUse_1_combine_0.8/`
- `results/archive/UCMerced_LandUse_120_combine_0.8/`
- `results/archive/UCMerced_LandUse_150轮_combine_0.8/`
- `results/archive/120轮fading信道0.8压缩率/`
